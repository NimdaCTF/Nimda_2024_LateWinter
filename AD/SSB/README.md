## Решение:
### Приватный флаг:  
Для получения доступа к флагу необходимо найти шелл. Найти его не трудно, нужно загрузить его в любое задание и понять, что отдача файлов веб-сервером настроена не правильно.  
Пример шелла:  

    <?PHP 

    phpinfo();

    ?>
